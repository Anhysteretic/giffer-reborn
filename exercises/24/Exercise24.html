Exercise24: In this exercise you will be turning on two LEDs at a time, sequencing from bottom to top.

ACTIVITIES:

  1. Copy the program from Starting Point Exercise24 into the Arduino IDE.
  2. Compile and upload this program to the Arduino and verify that LED’s 2 through 8 blink on and off with a 200ms delay between each, this then repeats four times, then the program pauses for 4 seconds and the whole thing starts over again….
  3. Modify the program so that LED’s 2 and 9 blink on and off together, then 3 and 10 together, then 4 and 11 together and so on up through LED’s 8 and 15 together. With a 100ms delay between each. Then repeat this sequence five times. Then the program pauses for 4 seconds and the whole thing starts over again….
    • Use one for loop for LED’s 2 & 9, 3 & 10, 4 & 11, and so on up through LED’s 8 & 15.
      • In this for loop try using the math otherLedNum=ledNum+7; to turn on the other LED.
    • Use a second for loop to repeat the entire sequence five times.
  4. Make Sure you are using variables in your delays. For example: delay(waitTime); instead of delay(100);.
  5. Save the completed sketch as YourInitials_Exercise24 in: Documents – Arduino – Exercises
  6. Copy the code into the Giffer website
  7. Enter your name and exercise number.
  8. Select the Run and Grade button.
  9. Look at the resulting Output image and make sure it is correct (Correct at lower part of image)
  10. If it is Correct -  use theCopy Page button to copy the evidence that you have correctly completed the exercise to clipboard.
  11. Go to CL 10.3.24 Upload Completed Exercise24:  And upload evidence that you have completed Exercise24
  12. Click "Prepare an answer" on NEO
  13. Click the "<>" (Source code) button.
  14. Paste by pressing CTRL + V
  15. Click "Save and submit for grading"
  16. If you made changes to the code in Giffer, then Copy the code from Giffer and paste it into the Arduino and re-save it as: YourInitials_Exercise24 - in your Documents/Arduino/Exercises Folder.
