Exercise15: Blinking on / off two LEDs at a time.

 

 

NEW:  Using more advanced mathematical expressions with variables;

 

DESCRIPTION: use one variable plus a constant to arrive at the value of another variable.

 

Example: otherLedNum = ledNum + 7

                        int ledNum;

                        int otherLedNum;

                        ledNum = 5; 

                        Other valid options include:

                        otherLedNum = ledNum - 2; // otherLedNum is storing 3

                        otherLedNum = ledNum * 2; // otherLedNum is storing 10

ACTIVITIES:

  3. Create another variable called otherLedNum.
  4. Create another variable otherLedNum = ledNumber +7.
  5. Modify the while loop to blink LEDs 2 – 8 on / off.
  6. Now modify the while loop to turn on LED 2 & 9 at the same time.
  7. Hint - Use something like the following two commands in order:
    • digitalWrite(ledNumber, HIGH);
    • digitalWrite(otherLedNum, HIGH);
