Exercise106: Using nested “for” loops to repeat a sequence four times.

ACTIVITIES:

  1. Copy the program from Starting Point Exercise106 into the Arduino IDE.
  2. Compile and upload this program to the Arduino and verify that LED’s 8 through 2 blink on and off in order with a 100ms delay between each four times, then the program pauses for 4 seconds and the whole thing starts over again….
  3. Modify the program so that LED’s 8 through 2 blink on and off in order with a 100ms delay between each ten times instead of four times…., then the program pauses for 4 seconds and the whole thing starts over again....
  4. Replace delay(100) with delay(waitTime);.
  5. Replace delay (4000) with an appropriate delay(waitTime*something).
  6. Save the completed sketch as YourInitials_Exercise106 in: Documents – Arduino – Exercises
  7. Copy the code into the Giffer website
  8. Enter your name and exercise number.
  9. Select the Run and Grade button.
  10. Look at the resulting Output image and make sure it is correct (Correct at lower part of image)
  11. If it is Correct -  use theCopy Page button to copy the evidence that you have correctly completed the exercise to clipboard.
  12. Go to CL 10.3.21 Upload Completed Exercise106:  And upload evidence that you have completed Exercise106
  13. Click "Prepare an answer" on NEO
  14. Click the "<>" (Source code) button.
  15. Paste by pressing CTRL + V
  16. Click "Save and submit for grading"
  17. If you made changes to the code in Giffer, then Copy the code from Giffer and paste it into the Arduino and re-save it as: YourInitials_Exercise106 - in your Documents/Arduino/Exercises Folder.
