Exercise29: In this exercise you will be blinking in reverse. This means you will start with all the LEDs on. Then blink them one at a time by turning the LED off, then turning the LED back on.

 

ACTIVITIES:

  1. Start with a blank sketch in the Arduino IDE and create a for loop in setup that tells all the pins connected to LEDs 2-15 to be outputs.
  2. Next create a for loop in loop that turns on all the LEDs at the same time. (Don’t use a delay inside this for loop).
  3. Next create another for loop that turns LED2 OFF then ON, and LED3 OFF then on ON and so on until you get to pin 15.
  4. Next create a for loop to repeat this sequence ten times.
  5. After doing the sequence ten times, create a for loop that turns OFF all the LEDs at the same time. (Don’t use a delay inside this for loop).
  6. Delay four seconds and then start at the beginning again.
  7. Make Sure you are using variables in your delays. For example: delay(waitTime); instead of delay(100);.
  8. Save the completed sketch as YourInitials_Exercise29 in: Documents – Arduino – Exercises
  9. Copy the code into the Giffer website
  10. Enter your name and exercise number.
  11. Select the Run and Grade button.
  12. Look at the resulting Output image and make sure it is correct (Correct at lower part of image)
  13. If it is Correct -  use theCopy Page button to copy the evidence that you have correctly completed the exercise to clipboard.
  14. Go to CL 10.3.29 Upload Completed Exercise29:  And upload evidence that you have completed Exercise29
  15. Click "Prepare an answer" on NEO
  16. Click the "<>" (Source code) button.
  17. Paste by pressing CTRL + V
  18. Click "Save and submit for grading"
  19. If you made changes to the code in Giffer, then Copy the code from Giffer and paste it into the Arduino and re-save it as: YourInitials_Exercise29 - in your Documents/Arduino/Exercises Folder.
