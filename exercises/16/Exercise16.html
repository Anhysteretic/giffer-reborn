Exercise16: Using two while loops to blinking on / off two LEDs at a time, count up then back down.

 

ACTIVITIES:

  1. Copy the program from your completed Exercise15 into the Arduino IDE.
  2. Create another while loop similar to the first that counts down from 7 – 3.
  3. Save the completed sketch as YourInitials_Exercise16 in: Documents – Arduino – Exercises
  4. Copy the code into the Giffer website
  5. Enter your name and exercise number.
  6. Select the Run and Grade button.
  7. Look at the resulting Output image and make sure it is correct (Correct at lower part of image)
  8. If it is Correct -  use theCopy Page button to copy the evidence that you have correctly completed the exercise to clipboard.
  9. Go to CL 10.3.16 Upload Completed Exercise16:  And upload evidence that you have completed Exercise16
  10. Click "Prepare an answer" on NEO
  11. Click the "<>" (Source code) button.
  12. Paste by pressing CTRL + V
  13. Click "Save and submit for grading"
  14. If you made changes to the code in Giffer, then Copy the code from Giffer and paste it into the Arduino and re-save it as: YourInitials_Exercise16 - in your Documents/Arduino/Exercises Folder.
