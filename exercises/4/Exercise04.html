Exercise04: First turn the LED on, then turn it off.

 

NEW:  digitalWrite(2,LOW);

DESCRIPTION: Tells Arduino to STOP sending power to pin 2.

ACTIVITIES:

  1. Copy the program from Starting Point Exercise04 into the Arduino IDE.
  2. Compile and upload this program to the Arduino and verify that LED 2 turns on for 1/2 second ahd then OFF for 5 seconds and repeats over and over .....
  3. Modify the program so that LED 2 turns ON for a 1/2 second, then OFF for a 1/2 second THEN LED 2 turns ON for a 1/2 second, then OFF for a 1/2 second (OR BLINKS TWICE) and THEN OFF for 5 seconds and repeats BLINKING in this pattern over and over……
  4. Modify the program so that LED 2 BLINKS ON and OFF THREE times (going ON for 1/2 second, then OFF for a 1/2 second). After the third BLINK  LED2 is OFF for 5 seconds and repeats BLINKING in this pattern over and over…...
  5. Modify the program so that LED 3 BLINKS ON and OFF THREE times like step 3 above.
  6. Save the completed sketch as YourInitials_Exercise04 in: Documents – Arduino – Exercises
  7. Copy the code into the Giffer website
  8. Enter your name and exercise number.
  9. Select the Run and Grade button.
  10. Look at the resulting Output image and make sure it is correct (Correct at lower part of image)
  11. If it is Correct -  use the Copy Page button to copy the evidence that you have correctly completed the exercise to clipboard.
  12. Go to CL 10.3.04 Upload Completed Exercise04:  And upload evidence that you have completed Exercise04
  13. Click "Prepare an answer" on NEO
  14. Click the "<>" (Source code) button.
  15. Paste by pressing CTRL + V
  16. Click "Save and submit for grading"
  17. If you made changes to the code in Giffer, then Copy the code from Giffer and paste it into the Arduino and re-save it as: YourInitials_Exercise04 - in your Documents/Arduino/Exercises Folder.
