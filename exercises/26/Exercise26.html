Exercise26: Use a for loop to blink the LEDs one at a time, in this order: 2,9,3,10,4,11,5,12,6,13,7,14,8,15.

ACTIVITIES:

  1. Copy the program from Starting Point Exercise26 into the Arduino IDE.
  2. Compile and upload this program and verify that LED’s 2 through 8 blink on and off in order with a 100ms delay between each four times, then the program pauses for 4 seconds and the whole thing starts over again….
  3. Modify the program so that LED’s blink in the following order 2,9,3,10,4,11,5,12,6,13,7,14,8,15.
    • Make sure you are using two for loops.
    • The outer for loop counts from 1-4.
    • The inner for loop counts from 2 – 8.
    • Change it so that the whole thing repeats five times instead of four times, then the program pauses for 4 seconds and the whole thing starts over again.
  4. Make Sure you are using variables in your delays. For example: delay(waitTime); instead of delay(100);.
  5. Save the completed sketch as YourInitials_Exercise26 in: Documents – Arduino – Exercises
  6. Copy the code into the Giffer website
  7. Enter your name and exercise number.
  8. Select the Run and Grade button.
  9. Look at the resulting Output image and make sure it is correct (Correct at lower part of image)
  10. If it is Correct -  use theCopy Page button to copy the evidence that you have correctly completed the exercise to clipboard.
  11. Go to CL 10.3.26 Upload Completed Exercise26:  And upload evidence that you have completed Exercise26
  12. Click "Prepare an answer" on NEO
  13. Click the "<>" (Source code) button.
  14. Paste by pressing CTRL + V
  15. Click "Save and submit for grading"
  16. If you made changes to the code in Giffer, then Copy the code from Giffer and paste it into the Arduino and re-save it as: YourInitials_Exercise26 - in your Documents/Arduino/Exercises Folder.
