Exercise27: Use a for loop to repeat the shown sequence 10 times. Alternate turning on all the red LEDs, then all the green LEDs.

ACTIVITIES:

  1. Copy the program from Starting Point Exercise27 into the Arduino IDE.
  2. Compile and upload this program and verify that LED’s 2 and 9 blink on and off , then LED’s 8 and 15 blink on and off with a 100ms delay between each four times, then the program pauses for 4 seconds and the whole thing starts over again….
  3. Modify the program so that all the red LEDs blink and then all the green LEDs blink, repeated ten times..
    • This sequence is much too complicated to do with a loop. Use digitalWrite(2,HIGH) like the example in starting point 27.
  4. Make Sure you are using variables in your delays. For example: delay(waitTime); instead of delay(100);.
  5. Save the completed sketch as YourInitials_Exercise27 in: Documents – Arduino – Exercises
  6. Copy the code into the Giffer website
  7. Enter your name and exercise number.
  8. Select the Run and Grade button.
  9. Look at the resulting Output image and make sure it is correct (Correct at lower part of image)
  10. If it is Correct -  use theCopy Page button to copy the evidence that you have correctly completed the exercise to clipboard.
  11. Go to CL 10.3.27 Upload Completed Exercise27:  And upload evidence that you have completed Exercise27
  12. Click "Prepare an answer" on NEO
  13. Click the "<>" (Source code) button.
  14. Paste by pressing CTRL + V
  15. Click "Save and submit for grading"
  16. If you made changes to the code in Giffer, then Copy the code from Giffer and paste it into the Arduino and re-save it as: YourInitials_Exercise27 - in your Documents/Arduino/Exercises Folder.
